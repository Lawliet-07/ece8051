# ece8051
## Getting started 
1. Clone the repository 
```bash
git clone https://github.com/sciduck/ece8051.git
```
2. Open Keil µVision5 
3. Project > Open Project
2. Project > New uVision Project...
3. Navigate to ece8051
4. Choose <b>8051.uvproj</b>
